# 产品需求文档（PRD）：盘串 · 每日心迹禅修

**版本**：V1.0 | **状态**：初稿 | **目标市场**：美国 iOS 应用商店

---

## 一、产品概述

### 1.1 产品名称
**盘串**（英文名：*Beads*）

名称来源于“盘手串”这一兼具仪式感与日常陪伴感的行为——既呼应禅宗念珠的修行传统，又带有年轻人“盘”一件心爱之物的活泼与亲切，剥离严肃宗教色彩，强调“日日摩挲、渐渐温润”的内心修养过程。

### 1.2 产品定位
> **“一个以禅宗智慧为底色、以每日心迹记录为锚点、以精美分享卡片为出口的轻量级精神修养工具。”**

它不是又一个冥想App，而是一个“每日参禅 + 记录心迹 + 广结善缘”的三阶闭环产品。它帮助用户在忙碌生活中每日获得片刻宁静，记录内心感悟，并与同频人建立轻量、温暖的精神连接。

### 1.3 核心理念（产品价值观）
1. **日拱一卒**：每日一参、一录，微行动带来内心渐变
2. **反焦虑**：不设排行榜、不比拼功德，只向内求
3. **轻社交**：陪伴而非表演，随喜而非攀比
4. **智慧不宗教**：禅宗哲学为底色，生活方式为外衣

### 1.4 一句话打动用户
> “每天3分钟，盘一串属于你自己的内心念珠。”

## 三、核心功能需求（三阶闭环详细拆解）

### 第一阶：参——每日智慧触碰

| 功能模块 | 详细需求 | 优先级 |
| :--- | :--- | :--- |
| **每日智慧推送** | 每日早上8:00（用户可自定义）推送一条“禅宗金句转译+今日微行动”，内容约40～80字 | P0 |
| **“我已参”交互** | 用户阅读后点击“我已参”，记录当日参悟完成；仅需一键，门槛极低 | P0 |
| **参悟连续记录** | 连续参悟天数可视化呈现为“念珠”形态——每7天成珠一颗，每颗珠子颜色随连续天数渐变（由浅至深） | P0 |
| **错过补参机制** | 允许用户补签最近3天的参悟（每日限补1次），维持连续记录；补签需完成简短的反思练习（如“今天有什么让你感到平静的瞬间？”） | P1 |
| **历史参悟回顾** | 用户可按日历视图查看所有参悟记录，已参悟日期有标记 | P1 |

**内容库规格**：
- 上线时需预制365条“金句+微行动”（至少1年量）
- 金句风格示例：
  - ✦ “你无法阻止海浪，但可以学会冲浪。” → 微行动：今天堵车/排队时，观察呼吸3次
  - ✦ “水不争而利万物。” → 微行动：在第一个会议前安静观察呼吸3次
  - ✦ “烦恼即菩提。” → 微行动：今天遇到一件烦心事，写下“它教会了我什么”
- 内容来源：禅宗公案、老子/庄子、慧能、铃木俊隆等，经现代语言转译
- 不可直接引用佛经原文；不出现“业力”“轮回”“供养”等宗教术语

### 第二阶：录——每日心迹留存

| 功能模块 | 详细需求 | 优先级 |
| :--- | :--- | :--- |
| **心得记录** | 用户可记录“今日心得短语”，限60字以内，保持轻量 | P0 |
| **心情标记** | 同时可点选当日心情：😌（平静）/ 😤（烦躁）/ 🙏（感恩）/ 😢（低落）/ 🔥（焦虑）/ 🌱（成长），选1～2个 | P0 |
| **“心迹集”归档** | 所有记录按月份归档为“心迹集”，每月自动生成一张合集封面（含当月情绪关键词云） | P1 |
| **导出为小书** | 用户可将任意月份的“心迹集”生成为图文卡片序列，可保存为图片或PDF | P2 |
| **记录自由** | 用户也可选择只标记心情不写文字，或直接跳过记录（仅完成参悟），不强制 | P0 |

### 第三阶：缘——广结善缘的轻社交

> **核心设计原则**：所有社交功能均为“匿名化 + 低打扰 + 积极向”，避免社交焦虑和攀比。

| 功能模块 | 详细需求 | 优先级 |
| :--- | :--- | :--- |
| **“我的菩提树”（个人精神主页）** | 展示：参悟总天数、念珠进度（含每颗珠子的日期标记）、本月情绪概览（可视化图表）、最近一条公开心得；可设置三级可见权限（公开/仅共修圈/仅自己） | P0 |
| **共修圈（轻社群）** | 用户可加入1～3个共修圈（如“职场禅”“父母静心”“创意空性”等）；圈内推送“今日共修主题”；圈内唯一互动：“随喜点灯”——看到他人的心得可点一盏灯，灯数累积在心得旁；所有成员显示为“匿名+随机禅意名称”（如“静水”“行云”）；不设评论区，不设私信 | P0 |
| **“智慧涟漪”（分享卡片）** | 用户可一键将任何一条心得生成为精美卡片；5～8种禅意视觉模板（水墨风/极简白/渐变夕阳等）；卡片底部标注“来自 盘串 Beads App”；分享至iOS系统分享面板（Instagram/Twitter/TikTok/微信等） | P0 |
| **“善缘树”（邀请机制）** | 每个用户有一棵“善缘树”；邀请朋友通过分享链接下载App；朋友完成7天参悟后，树上结一颗“善果”；善果仅作为“精神影响力”的可视化展示，不兑换任何实物或虚拟奖励；可查看“我的善缘”列表（受邀者匿名+完成天数） | P1 |
| **“每日共修主题”** | 每日App内推送一个全平台统一的“共修主题”（如“今天，接纳一件无法改变的事”），用户的心得可标记“参与共修”；在共修圈内可看到参与同主题的匿名心得集锦 | P1 |

### 跨阶通用功能

| 功能模块 | 详细需求 | 优先级 |
| :--- | :--- | :--- |
| **用户系统** | 邮箱/Apple ID/Google账号登录，支持Guest模式（本地存储，登录后同步） | P0 |
| **数据同步** | 跨设备同步（iCloud或自有BaaS），确保iPhone/iPad数据一致 | P1 |
| **推送通知** | 可自定义推送时间（默认8:00），支持“今日智慧”推送；“连续参悟提醒”（当天未完成时，20:00发送温和提醒） | P0 |
| **隐私与数据** | 支持一键导出所有个人数据（GDPR/CCPA合规）；用户可随时删除账户及所有数据 | P0 |

## 四、非功能性需求

## 五、运营与内容需求

### 5.1 内容生产体系
- **内容团队（初期可由创始人兼任）**：1～2人负责每日金句选编与微行动设计
- **内容来源策略**：
  - 70% 经典转译（禅宗公案/道家经典/铃木俊隆等）
  - 20% 现代正念改编（卡巴金/一行禅师等，确认版权OK）
  - 10% 用户投稿精选（未来可作为社区功能）
- **内容审核标准**：不涉及宗教传教；不涉政治；不医疗宣称；不含敏感词

### 5.2 种子用户获取（MVP阶段）
- Product Hunt 首发
- 在 r/Meditation、r/Buddhism、r/Mindfulness、r/Zen 等Subreddit发布（遵守社区规则）
- 制作“7天放下焦虑”免费邮件课程引流至App
- 联系粉丝量10万～50万的禅修/正念类YouTuber/TikToker，提供终身免费码换取评测

### 5.3 增长引擎（增长阶段）
- **UGC裂变**：“智慧涟漪”分享卡片自带App水印，形成社交传播
- **“善缘树”邀请**：无需物质奖励，用“精神影响力”驱动邀请
- **节日营销**：心理健康月（5月）、新年季（1月）、感恩节等节点做主题内容

### 5.4 用户留存策略
- **第3天**：推送“你已经参悟3天了，第一颗珠子正在成形”
- **第7天**：推送“恭喜！你的第一颗念珠已圆满”+ 生成“首周心迹卡”
- **第30天**：推送“你已经参悟一个月了，看看你的心迹集吧”+ 月度报告
- **流失召回**：连续3天未参悟，推送“念珠还在等你，回来继续盘吧”+ 一条精选金句

## 八、风险与规避

| 风险类型 | 具体描述 | 应对策略 |
| :--- | :--- | :--- |
| **内容风险** | 禅宗内容可能被误解为宗教传教，或引发文化挪用争议 | 所有文案强调“智慧”“哲学”而非“信仰”；内容注明“灵感源自东方哲学传统”；考虑邀请学术顾问背书 |
| **审核风险** | App Store对宗教类应用审核较严 | 提交时分类选择“生活方式”而非“宗教”；所有截图和描述不出现宗教符号（佛像、经文、僧侣等）；明确App不涉及医疗宣称 |
| **竞争风险** | 美区正念/冥想类App竞争激烈 | 差异化定位明确——“禅宗智慧+生活场景+轻社交”，非冥想音频类，避开正面竞争 |
| **留存风险** | 用户新鲜感过后流失 | 核心机制设计（念珠可视化）本身就是留存引擎；结合推送策略和月度报告强化“成长感” |
| **合规风险** | CCPA/GDPR数据隐私合规 | 使用Firebase/Supabase等合规BaaS；提供数据导出和删除功能；隐私政策明确清晰 |

## 十、附录：关键界面描述（UX方向参考）

| 界面 | 核心元素 | 视觉风格建议 |
| :--- | :--- | :--- |
| **今日首页** | 上方：今日金句卡片（可点击“我已参”）；中部：今日微行动；下方：快速记录入口（心情点选 + 一句话输入） | 大面积留白，柔和暖色（米白/淡灰/浅金），字体偏大，营造“静心”感 |
| **念珠进度页** | 横向滚动的念珠视图，每颗珠子代表7天，当前珠子高亮并微微发光；下方显示总参悟天数 | 珠子为磨砂质感，渐变暖色，背景深色（夜空感） |
| **心迹集页** | 按月排列的时间线，每条心得以卡片形式展示，心情图标显眼 | 细腻阴影，卡片圆角大，排版接近极简笔记本风格 |
| **“我的菩提树”主页** | 顶部：参悟天数 + 念珠进度徽章；中部：情绪周报小图表；下方：最近三条公开心得的预览 | 个人主页感，清晰的信息层级，可设置公开/私有开关 |
| **共修圈页** | 顶部显示圈名和今日共修主题；中部为匿名心得流，每条心得下方显示“收到X盏灯”；底部为“发布心得参与共修”按钮 | 温和的卡片流，匿名名称用柔和灰字，灯图标为暖黄色小圆点 |
| **分享卡片模板** | 5～8种风格：水墨禅意、极简白、渐变色、羊皮纸质感、暗夜星空等；文字居中或左下排布，整体留白>60% | 适合Instagram/Twitter的方形比例（1:1）或Story比例（9:16），品牌水印低调不显眼 |
